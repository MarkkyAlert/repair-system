## [1.0.0] — เผยแพร่ครั้งแรก

รุ่นแรกของระบบแจ้งซ่อม/บำรุงรักษา (source-code template) ครบวงจร:

- **วงจรใบงาน** — แจ้ง → อนุมัติ → มอบหมายช่าง → รับงาน/เริ่ม/สรุปผล → ผู้แจ้งยืนยันปิดงาน + ให้คะแนน 1–5; เปิดงานซ้ำ (reopen) / ยกเลิก / ทำซ้ำ
- **ช่องทาง Guest ผ่าน QR** — แจ้งปัญหาโดยไม่ต้องล็อกอิน + หน้าติดตามสถานะ (`/track`); แอดมินแปลงเป็นใบงาน/ปฏิเสธ
- **สิทธิ์ 4 บทบาท** — requester / manager / technician / admin (แยกหน้าที่อนุมัติ)
- **ทรัพย์สิน + QR** — ทะเบียนทรัพย์สิน, สร้าง/พิมพ์ QR, นำเข้าผ่าน CSV
- **รายงาน 9 แบบ** — ภาพรวมผู้บริหาร, SLA, ผลงานช่าง, จุดเสียบ่อย, แนวโน้ม, งานค้าง, เปิดซ้ำ, CSAT, ความน่าเชื่อถือทรัพย์สิน (ส่งออก CSV / Excel / PDF รองรับภาษาไทย)
- **งานเบื้องหลัง (cron)** — คิด SLA เกินกำหนด + คิวอีเมลแจ้งเตือน + สำรองฐานข้อมูล
- **ตั้งค่าแบรนด์ผ่านหน้า admin** — ชื่อระบบ, โลโก้, ธีมสี; นำเข้าผู้ใช้ผ่าน CSV; จัดการเทมเพลตอีเมล
- **ติดตั้งง่ายสำหรับ shared hosting** — Setup Wizard, `check-requirements.php`, สำรองฐานข้อมูลผ่านหน้า admin (ไม่ต้องมี shell)

> อัปเดตไฟล์นี้ทุกครั้งที่ปล่อยเวอร์ชันใหม่ให้ผู้ซื้อ — สรุปสิ่งที่ **เพิ่ม (Added) / แก้ (Fixed) / เปลี่ยน (Changed) / เลิกใช้ (Removed)** พร้อมเลขเวอร์ชันในไฟล์ `VERSION`
